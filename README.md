🚀 React Business Website

A modern and responsive business website built using **React.js**.
This project demonstrates component-based architecture, navigation, and basic form handling using React hooks.

📌 Features

🔹 Component-based structure
🔹 Responsive layout (basic)
🔹 Smooth navigation (anchor scroll)
🔹 Dynamic rendering using JavaScript map()
🔹 Contact form with useState hook
🔹 Button click event handling
🔹 Clean and modern UI design

🧱 Project Structure

```
src/
 ┣ components/
 ┃ ┣ Navbar.jsx
 ┃ ┣ Hero.jsx
 ┃ ┣ MarqueeText.jsx
 ┃ ┣ Products.jsx
 ┃ ┣ About.jsx
 ┃ ┣ USP.jsx
 ┃ ┣ Contact.jsx
 ┃ ┗ Footer.jsx
 ┣ App.js
```

## ⚙️ Technologies Used

React.js
JavaScript (ES6)
HTML5
CSS3

## 🔥 Key Concepts Used

Functional Components
useState Hook
Event Handling (onClick, onSubmit)
Array Mapping for Dynamic UI
Basic Navigation using Anchor Links
